# 🧮 Multiplication App

A simple and interactive **Multiplication Practice App** built using **HTML, CSS, and JavaScript**.  
The app generates random multiplication problems, lets users input their answers, and keeps track of their score.  

🔗 **Live Demo**: [Multiplication App](https://vaibhavsain.github.io/Multiplication-App/)

---

## ✨ Features
- 🎲 **Random Multiplication Problems** generated every round.  
- ⌨️ **User Input Validation** to check correct/incorrect answers.  
- 🏆 **Score Tracking** to keep the practice engaging.  
- 🎨 **Clean & Responsive UI** built with CSS.  
- ⚡ **Lightweight & Fast** — no external libraries required.  

---

## 🛠️ Tech Stack
- **HTML5**  
- **CSS3**  
- **JavaScript (ES6)**
